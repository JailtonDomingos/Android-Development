git init - Para iniciar um branch local
git add <arquivo> para adicionar ao branch local
git status - Status
git commit -m "comment" - Para comitar no branch local

git remote add origin <servidor> - Fazer a ligação sendo origin o local que será usado para comitar no branch remoto
git push origin <branch remoto> - Para comitar o conteudo do branch atual(local) no Git hub

git checkout -b <nome> - Criar um novo branch local(Caso será a primeira inicialização de um branch local, é necessário colocar o "NOME" como o url.git).
git checkout master - Trocar de branch local(sendo "master" o nome do branch que quer dar switch)
git branch -d <nome> - Excluir branch
git push origin <nome> - O novo branch não estará disonivel para commit remoto, para isso adicione

git pull - Baixar para o branch local
git diff <branch origem> <branch destino> - differenciar branch local do branch selecionado
git merge <branch> - Fazer junção de branch's
